Esta pagina tiene como finalidad enseñar todas las funciones disponibles que dispone el sistema de base de datos MySQL

¿Que es MySQL?
  • MySQL es un sistema de administración de bases de datos relacionales
  • MySQL es ideal para aplicaciones grandes y pequeñas
  • MySQL fue lanzado por primera vez en 1995
¿Quienes usan MySQL?
  • Grandes sitios web como Facebook, Twitter, Airbnb, Booking.com, Uber, GitHub, YouTube, etc.
  • Sistemas de gestión de contenidos como WordPress, Drupal, Joomla!, Contao, etc.
  • Una gran cantidad de desarrolladores web en todo el mundo.
