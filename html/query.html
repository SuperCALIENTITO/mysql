Algunos comandos de interes
  • SELECT - Selecciona/Extrae informacion de una base de datos
  • UPDATE - Actualiza informacion de una base de datos
  • DELETE - Elimina informacion de una base de datos (No confundir con DROP)
  • INSERT INTO - Inserta/Añade informacion a una base de datos
  • CREATE DATABASE - Crea una nueva base de datos
  • ALTER DATABASE - Modifica una base de datos
  • CREATE TABLE - Crea una nueva tabla de datos
  • ALTER TABLE - Modifica una tabla de datos
  • DROP TABLE - Elimina una tabla de datos (No confundir con DELETE)
  • CREATE INDEX - Crea un indice (Llave de busqueda)
