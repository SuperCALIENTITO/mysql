La instrucción SELECT
La instrucción SELECT se utiliza para seleccionar datos de una base de datos. Los datos que se devuelven/entregan se almacenan en una tabla de resultados, denominada conjunto de resultados.
Ejemplos de Escritura
SELECT columna1, columna2, ...
FROM tabla_datos;
Selecciona las columnas columna1 y columna2 de datos de la tabla tabla_datos
SELECT * FROM tabla;
Selecciona todas las columnas dentro de la tabla tabla_datos
Ejemplos
Con el comando SELECT se puede seleccionar todos los datos de una tabla de la siguiente forma.
SELECT * FROM Clientes;
