La instrucción SELECT
El comando SELECT de MySQL es un comando utilizado para recuperar datos de una base de datos.
Este comando se utiliza generalmente junto con la cláusula WHERE para especificar qué registros se deben recuperar.
SELECT con WHERE
La sintaxis del comando SELECT es la siguiente:
SELECT
       [campo1, campo2, ...]
FROM
       [nombre de la tabla]
WHERE
       [condición];
SELECT con WHERE y ORDER BY
El comando SELECT puede utilizarse también con la cláusula ORDER BY para especificar el orden en el que se deben recuperar los registros.
La sintaxis del comando SELECT es la siguiente:
SELECT
       [campo1, campo2, ...]
FROM
       [nombre de la tabla]
WHERE
       [condición];
SELECT con WHERE y LIMIT
El comando SELECT también se puede utilizar con la cláusula LIMIT para especificar el número máximo de registros que se deben recuperar.
La sintaxis del comando SELECT con la cláusula LIMIT es la siguiente:
SELECT
       [campo1, campo2, ...]
FROM
       [nombre de la tabla]
LIMIT
       [número máximo de registros];
Ejemplos
Con el comando SELECT se puede seleccionar todos los datos de una tabla de la siguiente forma.
SELECT * FROM Clientes;
